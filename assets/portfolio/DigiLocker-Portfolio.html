DigiLocker - Smart Locker Application

Project information

  • Category: Application Development
  • Client: Kellanova
  • Project date: June 10th, 2024

The DigiLocker Portal is a React front-end with a Node.js(Express) back-end. This web application was built for an iPad that will server as a one-stop solution for employees engaging with a Smart Locker system at specific locations. The application features a component based architecture with modern React hooks, specifically the useState and useEffect hooks.

Users will sign in to the application, and an animation will play to sign them in quickly, and the home screen will show (the first image in the slideshow). Administrators of the DigiLocker application have a custom-built admin menu that will allow them to manage all sorts of features that were previously only available on a web UI from the vendor of the Smart Locker.

There are numerous custom endpoints built to allow the front-end and back-end to communicate seamlessly thanks to Express.js. There are also many custom functions created in the interest of end-to-end automation. This application hosts a complete API solution for working with the Smart Locker. The application also communicates directly with Graph API so that it works in an enterprise environment for the client.
